# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.x.    | :white_check_mark: |
| 4.x     | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Please raise any vulnerabilities as issues and tag them with `vulnerability`. I will ensure to respond within a couple of days.
